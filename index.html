Git과 Github 내용 정리

1. Git 과 Github 의 차이점


git = 음악편집기
github = 자작 노래 게시판

2. init, add, commit -m

init : 디렉토리를 git 레포지토리로 만들어 디렉토리 내의 파일을 git이 추적하게 한다. untracked 상태

add : 을 스테이징한다 untracked 상태 -> staged 상태

commit -m : 스테이징된 파일들을 커밋하여 하나의 버전으로 만들어 git 데이터베이스에 저장한다. staged 상태 -> committed 상태

3. remote, clone, push, pull

remote : 이라는 별칭으로 의 원격 레포지토리를 등록한다.

clone : 의 원격 레포지토리를 가져와서 디렉토리에 복제한다.

push : 의 원격 레포지토리로 변경사항을 업데이트한다.

pull : : 의 원격 레포지토리를 가져온 후 에 병합한다.

4. Git Push Pull Commands

5. branch, checkout

branch란 포인터다 평행세계를 만들어 원래의 세계에서 평행세계로 넘어가서 그 세계에서 마음껏 실험적인 행동을 자행하는것

checkout이랑 브랜치로 이주하는 것이다

6. merge 2가지 설명 (fast-forward/Merge conflict)

fast-forward : git이 단순히 master 브랜치가 testing 브랜치가 포인팅하고 있는 커밋을 포인팅하게 하는것

Merge conflict : 병합할 때 발생하는 충돌

잠을 제대로 못 자면 일어나는 7가지 문제점

  1. 고혈압 - 먼저 수면 부족, 혹은 불면증은 심혈관 건강에 영향을 미칠 수 있다.

    이 문제는 혈압 상승으로 이어져 심장이 필요 이상으로 작동하게 만들 수 있다.

    하루에 7시간 이하로 자는 사람은 보통 심장 수축 혈압이 최대 123까지 상승한다. 이상적으로는 120 이하여야 한다.

  2. 식욕 증가 - 불면증이 체중 증가와 관련이 있는 이유 중 하나는 휴식을 취하는 동안 식욕이 사실 증가하기 때문이다.

    잠을 제대로 못 자면 렙틴 수치가 높아진다. 이것은 충분히 먹었을 때 식욕을 조절하는 역할을 하는 호르몬이다.

    잠을 못 자면 오히려 그렐린이라는 식욕 촉진 호르몬이 생성되어 필요한 양보다 더 먹게 만든다.

  3. 에너지 소비 감소 - 잠을 충분히 자지 못하면 신진대사에 직접 영향이 가고, 피로를 느끼기 때문에 에너지 소비량도 줄어든다.

    수면 문제가 있는 사람은 신체적으로 덜 활동적인 편이다. 그래서 지방 및 과다 열량이 쌓이게 만든다.

  4. 집중력 저하 - 또한 잠을 제대로 자지 못했을 때 가장 큰 문제를 겪는 부위는 바로 뇌다.

    뇌는 낮 동안 활동성이 떨어지며, 피로 때문에 집중하지 못하고 정상적인 인지 과정을 수행할 수 없게 된다.

    시간이 지나면서 잠을 제대로 못 자는 것 때문에 뇌가 플라크 형성 단백질을 제거하고 알츠하이머와 같은 만성 질환을 유발하는 것을 막는 역할이 영향을 받을 수 있다.

  5. 면역력 약화 - 게다가 수면 부족으로 인한 신체적 피로는 면역력에 직접 영향을 미칠 수 있다. 이는 면역계의 기능을 감퇴시키고, 호흡기 질환 및 감염증의 위험을 높인다.

    항체 생성 감소는 바이러스, 박테리아, 곰팡이가 더 쉽게 공격하도록 만든다. 그래서 잠을 제대로 못 자면 병에 걸리는 것이 드문 일이 아니다.

  6. 스트레스 및 신경과민 -잠을 제대로 못 자는 것이 정신 건강에 미치는 영향을 거의 즉시 나타낸다. 그래서 잠을 제대로 못 잔 후에 스트레스를 받고 신경과민이 되는 것이 이상한 일이 아니다.

    충분히 휴식을 취하지 못하는 것은 세로토닌 및 엔도르핀 생성을 낮추는데, 이 두 가지 호르몬은 행복감을 증진하는 호르몬으로 알려져 있다.

    이 경우 오히려 부정적인 감정 반응을 유발하는 물질들인 아드레날린과 코르티솔의 생성이 늘어나게 된다.

  7. 근육 약화 -휴식을 취하는 동안 근육은 다음 날의 활동을 위해 자신을 재생하기 위한 이완의 상태로 들어간다.

    동시에 성장 호르몬이 분비되는데, 이런 호르몬들은 근육량을 늘린다. 잠을 제대로 못 자면, 이런 과정들이 없어지면서 근육이 약해지고 피로해지게 된다.

    잠을 제대로 자지 않고 몸을 너무 과하게 쓰는 것은 심각한 신체적 과로로 이어질 수도 있다.

    잠을 충분하게 제대로 못 잤을 때는 헬스장을 가거나 고강도의 운동을 하는 것을 피해야 한다.

    잠을 자는 데 문제가 있는가? 그렇다면 잠을 잘 자는 데 도움이 되는 습관을 기르는 것이 좋다.

    잠을 제대로 못 자도 그 영향이 없는 것처럼 느낄 수 있다. 하지만 며칠이 지나면 이것이 몸에 영향을 미치기 시작할 것이며, 수면 부족의 결과가 나타나기 시작할 것이다.

잠을 안 자면 일어나는 4가지 좋은 점

  1. 알바월급 - 격일로 하루에 8시간 시급 7731원이다.

    7731 * 8 * 15일로 가정하였을때 927,720원을 벌 수 있다

    즉 쏠쏠하다.

  2. Flutter 학습 - 플러터를 방학때 프로젝트로 빠르게 습득할 수 있다.

    피곤하지만 엄청 유용해서 잠을 안자고 해도 재밌다

    즉 변태같다.

  3. NEWAVE 디자인 견문확장 - 뉴웨이브에서 디자인 적인 견문을 확장할 수 있다.

    하루 최소 2~3시간 회의로 늘 디자인을 고려하니 매우 견문이 넓어졌다

    즉 콘디도 재밌어보인다.

  4. IT 서비스 수업 - IT 서비스 수업으로 많은 툴과 기술을 단시간에 습득했다

    이제 정말 많은 툴을 이용해서 미리 서비스를 구현하는 것은 정말 쉬운 일인 것 같다.

    즉 아이디어 싸움이다.
